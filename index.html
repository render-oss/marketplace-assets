Marketplace Assets

This site is a simple static asset host for GitHub Marketplace images. Add image files to /images and Render will serve them at public URLs.

Public URL pattern

After deploy, any file placed in images/ will be available at:
https://<your-service>.onrender.com/images/<filename>

Suggested filenames

  • images/hero.png
  • images/pr-preview.png
  • images/architecture-diagram.png
  • images/dashboard.png
  • images/customer-logos.png

Current repo files

  • `github-marketplace.md`
  • `images/`
